To use in your reference manager


Mendeley

Use the button below to save your .csl file to the following location:

  • Windows Vista and Windows 7 

    C:\Users\<USER_NAME>\AppData\Local\Mendeley Ltd\Mendeley Desktop\citationStyles-1.0

  • Windows XP 

    C:\Documents and Settings\<USER_NAME>\Local Settings\Application Data\Mendeley Ltd\
    Mendeley Desktop\citationStyles-1.0

  • Linux 

    ~/.local/share/data/Mendeley Ltd./Mendeley Desktop/citationStyles-1.0

  • Mac OS-X 

    /Users/<USER_NAME>/Library/Application Support/Mendeley Desktop/citationStyles-1.0

Zotero

Use the button below to save your .csl file whereever you like, then:

  • Firefox plugin

    Drag the saved file into any open Firefox window.

  • Standalone

    In Zotero standalone go to Preferences->Cite tab->Styles, and click the '+' button to add your new style.
